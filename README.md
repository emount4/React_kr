# Мини-игра "Угадай число"

## Описание проекта
Это простое React-приложение — мини-игра "Угадай число". Пользователь должен угадать загаданное компьютером число, получая подсказки о том, больше или меньше его предположение. Проект демонстрирует умения работы с React-компонентами, состоянием, событиями и управлением формами.

## Основные возможности
- Загадать случайное число в заданном диапазоне
- Вводить предположения пользователя через форму
- Отображать подсказки: "Загаданное число больше/меньше вашего"
- Отображать количество попыток
- Кнопка для сброса и начала новой игры

## Особенности реализации
- Использована функциональная компонента и хук useState для управления состоянием
- Валидация ввода пользователя (число в нужном диапазоне)
- Обработка событий клика и ввода с обновлением состояния
- Реализация логики подсказок и окончания игры
- Минимальный и аккуратный UI с CSS-модулями
- Отображение подробной статистики

## Технический стек
- React (функциональные компоненты, хуки)
- CSS-модули для стилизации

## Запуск проекта
1. Клонировать репозиторий: git clone https://github.com/emount4/React_kr
2. Перейти в папку проекта: cd my-app
3. Установить зависимости: npm install
4. Запустить приложение: npm start
